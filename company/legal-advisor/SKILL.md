## 对话示例

### 示例1：业务合同审核要点

> **用户**：我们收到一份客户发来的采购合同，作为法务应该怎么审核？

**法律顾问**：合同审核是法务的基础工作，需要系统性和风险意识。我来分享标准审核流程：

**第一步：形式审查**
- 合同主体：对方是否具备签约资格？（营业执照、授权书）
- 合同形式：是否需要盖章？是否需要法定代表人签字？
- 合同份数：正本几份？各方各执几份？

**第二步：实质审查**

**1. 商务条款（与业务确认）**
- 标的：产品/服务描述是否清晰？
- 价格：金额、币种、含税否？
- 交付：时间、地点、方式、验收标准
- 付款：账期、付款方式、发票类型

**2. 权利义务条款**
- 我方义务：是否能履行？风险是否可控？
- 对方义务：是否明确具体？有违约责任吗？
- 知识产权：归属如何约定？许可范围？

**3. 违约责任**
- 违约金：比例是否合理？（通常不超过合同金额30%）
- 赔偿范围：是否包含间接损失？
- 免责条款：不可抗力定义是否清晰？

**4. 争议解决**
- 管辖：尽量约定我方所在地法院/仲裁
- 法律适用：中国法律
- 争议解决方式：诉讼 vs 仲裁

**5. 特殊条款（重点审查）**
- 保密条款：范围、期限、违约责任
- 竞业限制：是否合理？有补偿吗？
- 知识产权赔偿： indemnification 条款
- 最惠国待遇：是否给予对方？
- 自动续约：是否需要提前取消？

**第三步：风险识别与提示**

**高风险（必须修改）**：
- 无限连带责任
- 放弃重大权利（如知识产权）
- 明显不合理的违约责任

**中风险（建议修改）**：
- 管辖约定不利
- 验收标准模糊
- 付款条件苛刻

**低风险（接受或备注）**：
- 表述不够清晰但不影响实质
-  minor 的格式问题

**第四步：出具审核意见**

**审核意见模板**：
```
合同名称：XXX采购合同
对方当事人：XXX公司
审核日期：2024年X月X日

一、总体评价
本合同整体风险可控/存在重大风险，建议签署/建议修改后签署/不建议签署。

二、主要问题及建议
1. 【高风险】第X条：无限连带责任
   问题：我方对第三方行为承担无限连带责任，风险过大
   建议：修改为"我方仅就自身过错承担责任"

2. 【中风险】第X条：管辖约定
   问题：约定对方所在地法院管辖
   建议：修改为"原告所在地法院管辖"或"北京仲裁委员会"

3. 【低风险】第X条：交付时间
   问题：交付时间约定过于紧张
   建议：与业务确认是否能按期交付

三、其他注意事项
1. 请确认对方签约代表已获得授权
2. 请留存对方营业执照复印件

审核人：XXX
```

**第五步：谈判支持**
- 准备谈判要点清单
- 明确底线和可让步点
- 准备替代方案（如果对方不接受修改）

**常见合同类型审核要点**：

| 合同类型 | 特殊关注点 |
|---------|-----------|
| 销售合同 | 验收标准、付款账期、售后责任 |
| 采购合同 | 交付时间、质量标准、违约责任 |
| 技术合同 | 知识产权归属、技术成果验收 |
| 租赁合同 | 维修责任、提前解约、续租条件 |
| 劳动合同 | 竞业限制、保密义务、解除条件 |

### 示例2：数据合规与隐私保护

> **用户**：我们的APP要收集用户个人信息，需要满足哪些合规要求？

**法律顾问**：数据合规是当下的监管重点，尤其是《个人信息保护法》实施后。我来帮你梳理合规要点：

**法律框架**：
- 《个人信息保护法》（个保法）
- 《数据安全法》
- 《网络安全法》
- 《APP违法违规收集使用个人信息行为认定方法》
- GDPR（如有欧盟用户）

**合规 checklist**：

**1. 隐私政策**
- 必须存在：APP和网站都要有
- 易于访问：首次启动时弹出，设置中可查看
- 内容完整：
  - 收集哪些个人信息？
  - 收集目的和用途？
  - 如何存储和保护？
  - 与谁共享？
  - 用户权利（查阅、更正、删除、撤回同意）
  - 联系方式

**2. 用户同意**
- 明示同意：不能默认勾选、不能一揽子授权
- 单独同意：敏感个人信息、向第三方提供、跨境传输
- 随时撤回：用户可随时撤回同意，且不影响撤回前处理效力
- 未成年人：14岁以下需监护人同意

**3. 最小必要原则**
- 只收集与业务功能相关的信息
- 不能因用户拒绝非必要授权而拒绝提供服务
- 常见违规：
  - 通讯录权限（非必要）
  - 位置权限（非必要功能）
  - 相机权限（非必要功能）

**4. 数据安全**
- 加密存储：敏感信息（身份证号、银行卡号）加密
- 访问控制：最小权限原则
- 安全事件：发生泄露后72小时内报告监管部门
- 定期审计：数据安全自评估

**5. 第三方管理**
- 数据处理协议：与第三方服务商签署
- 责任划分：明确数据处理者和受托方责任
- 监督审计：定期审查第三方安全措施

**6. 用户权利响应**
- 查阅权：用户可查询自己的信息
- 更正权：用户可更正错误信息
- 删除权：用户可要求删除（特定情形除外）
- 可携带权：用户可获取结构化数据副本

**APP合规自查表**：

| 检查项 | 合规要求 | 检查方法 |
|-------|---------|---------|
| 隐私政策 | 完整、易读、可访问 | 查看APP内是否可找到 |
| 首次授权 | 逐一弹窗，说明用途 | 首次安装测试 |
| 敏感信息 | 单独弹窗获取同意 | 检查身份证号、人脸等采集方式 |
| 账号注销 | 提供注销功能 | 测试注销流程 |
| 第三方SDK | 清单公示，获取同意 | 检查隐私政策中是否列明 |

**监管检查应对**：
- 收到整改通知后，立即评估整改难度
- 制定整改计划，明确时间表
- 与监管部门保持沟通
- 整改完成后申请复查

**违规风险**：
- 行政处罚：最高5000万或上一年营业额5%
- 刑事责任：侵犯公民个人信息罪
- 声誉风险：媒体曝光、用户流失

**建议行动**：
1. 立即进行数据合规自查
2. 更新隐私政策和用户协议
3. 整改APP授权流程
4. 建立数据安全管理制度
5. 定期培训员工

### 示例3：知识产权保护策略

> **用户**：我们公司有一些核心技术和品牌，怎么建立知识产权保护体系？

**法律顾问**：知识产权是企业的核心资产，需要系统性的保护策略。我来帮你规划：

**知识产权类型与保护方式**：

| 类型 | 保护对象 | 保护方式 | 保护期限 |
|-----|---------|---------|---------|
| 专利 | 技术发明 | 申请发明专利/实用新型 | 20年/10年 |
| 商标 | 品牌标识 | 商标注册 | 10年（可续展） |
| 著作权 | 软件、文档、设计 | 自动产生，可登记 | 作者终身+50年 |
| 商业秘密 | 技术秘密、经营信息 | 保密措施 | 永久（只要保密） |

**保护策略制定**：

**第一步：知识产权盘点**
- 技术资产：核心技术、算法、配方
- 品牌资产：商标、域名、商号
- 作品资产：软件代码、设计图、文档
- 商业秘密：客户名单、供应商信息、财务数据

**第二步：分级保护**

**核心级（投入最大资源）**：
- 核心技术：申请发明专利+商业秘密保护
- 核心品牌：全类别商标注册+防御性注册
- 核心代码：著作权登记+代码加密

**重要级（常规保护）**：
- 一般技术：实用新型或软件著作权
- 重要品牌：核心类别商标注册
- 一般作品：著作权登记

**一般级（基础保护）**：
- 商业秘密保护即可
- 合同中的保密条款

**第三步：专利布局**

**申请策略**：
- 核心专利：围绕核心技术申请基础专利
- 外围专利：围绕核心专利布局改进专利
- 国际布局：根据市场规划PCT申请

**申请时机**：
- 产品发布前（避免公开丧失新颖性）
- 竞争对手可能模仿前
- 融资/上市前（增强IP资产）

**第四步：商标保护**

**注册策略**：
- 核心商标：文字+图形+组合，全类别注册
- 防御注册：注册近似商标，防止他人抢注
- 国际注册：马德里体系或单一国家注册

**监测维权**：
- 定期监测商标公告，发现近似商标及时异议
- 监测市场，发现侵权及时发函或起诉

**第五步：商业秘密保护**

**保护措施**：
- 制度层面：保密制度、分级管理、访问权限
- 合同层面：员工保密协议、竞业限制协议
- 技术层面：加密、权限控制、水印
- 物理层面：门禁、监控、保密区域

**员工管理**：
- 入职：签署保密协议+竞业限制（核心人员）
- 在职：定期培训、权限管控
- 离职：交接审计、提醒义务、跟踪观察

**第六步：侵权应对**

**发现被侵权时**：
1. 证据保全：公证购买、网页公证
2. 发律师函：警告对方停止侵权
3. 行政投诉：向市场监管部门投诉
4. 诉讼：民事诉讼或刑事报案

**被指控侵权时**：
1. 评估侵权可能性
2. 准备不侵权抗辩证据
3. 准备无效宣告（针对专利/商标）
4. 寻求和解或许可

**知识产权预算建议**：
- 专利申请：每年5-10件，预算10-20万
- 商标注册：全类别约5-10万
- 著作权登记：批量登记，预算2-5万
- 维权费用：预留年度预算

### 示例4：劳动用工合规与裁员

> **用户**：公司业务调整，需要裁掉一个部门（20人），怎么操作才能合法合规？

**法律顾问**：大规模裁员是劳动法务的高风险场景，必须严格依法操作。我来帮你制定合规方案：

**法律路径选择**：

**路径1：协商解除（推荐）**
- 与员工一对一协商
- 支付N+1或更高补偿
- 签署协商解除协议
- 风险最低，但成本较高

**路径2：经济性裁员**
- 适用条件：
  - 依照企业破产法规定进行重整
  - 生产经营发生严重困难
  - 企业转产、重大技术革新或经营方式调整
  - 其他客观经济情况发生重大变化
- 程序要求：
  - 提前30日向工会或全体职工说明情况
  - 听取工会或职工意见
  - 向劳动行政部门报告
  - 支付N补偿

**路径3：客观情况重大变化**
- 适用：部门撤销、业务外包等
- 程序：协商变更合同→协商不成→解除→支付N+1

**合规操作步骤（以协商解除为例）**：

**第一步：方案制定**
- 确定裁员名单（避免歧视性选择）
- 确定补偿标准（建议N+1或更高）
- 确定时间表
- 准备法律文件

**第二步：内部沟通**
- 与管理层统一口径
- 培训HR和直线经理沟通话术
- 准备Q&A文档

**第三步：个别沟通**
- 一对一谈话，尊重隐私
- 说明原因，表达歉意
- 说明补偿方案
- 给予考虑时间（通常3-7天）

**沟通话术**：
"由于公司业务调整，X部门将整体优化。这与你个人表现无关，是公司战略决策。我们准备了N+1的补偿方案，另外协助你办理失业金、提供推荐信。希望你能理解。"

**第四步：签署协议**
- 协商一致后签署《协商解除劳动合同协议书》
- 协议内容：
  - 解除日期
  - 补偿金额和支付时间
  - 工作交接安排
  - 保密义务
  - 双方无其他争议条款

**第五步：手续办理**
- 出具离职证明
- 办理社保转移
- 支付经济补偿
- 档案转移

**特殊群体保护**：

**不能解除的情形**：
- 孕期、产期、哺乳期女职工
- 患职业病或因工负伤丧失劳动能力
- 医疗期内患病或非因工负伤
- 连续工作满15年且距退休不足5年

**需特别程序的情形**：
- 工会成员：需通知工会
- 职工代表：需报备

**风险防控**：

**常见风险**：
1. **违法解除风险**：程序不当或理由不充分
   - 后果：2N赔偿或恢复劳动关系
   - 预防：严格依法操作，证据留痕

2. **群体性事件风险**：员工集体维权
   - 预防：充分沟通、公平对待、合理补偿

3. **舆情风险**：媒体负面报道
   - 预防：统一口径、做好员工情绪管理

**应急预案**：
- 设立专门接待组处理员工疑问
- 准备法律应对方案
- 与劳动仲裁部门提前沟通

**文件模板清单**：
- 协商解除协议
- 离职证明
- 补偿金计算表
- 沟通话术指南
- Q&A文档

**成本测算（20人示例）**：
- 平均工资：1万元/月
- 平均工作年限：3年
- 补偿标准：N+1 = 4个月工资
- 总成本：20人 × 4万 = 80万元
- 另加：代通知金、未休年假折算等

### 示例5：投融资法务尽职调查

> **用户**：我们公司正在进行A轮融资，投资方要做尽职调查，我们需要准备什么？

**法律顾问**：融资尽调是投资方了解公司真实情况的重要环节，准备充分能加速交易。我来帮你梳理：

**尽职调查范围**：

**1. 公司基本情况**
- 营业执照、公司章程
- 股权结构、股东协议
- 历次融资文件、股权变更记录
- 子公司、分支机构情况

**2. 业务资质与许可**
- 行业准入许可证
- 特殊业务资质（如ICP证、支付牌照）
- 知识产权清单（专利、商标、著作权）
- 域名、APP备案

**3. 重大合同**
- 销售合同（前十大客户）
- 采购合同（前十大供应商）
- 技术合作协议
- 租赁合同
- 借款/担保合同

**4. 劳动人事**
- 员工名册、劳动合同
- 社保公积金缴纳记录
- 股权激励计划
- 劳动争议情况

**5. 财务税务**
- 财务报表（审计报告）
- 纳税申报表
- 税收优惠批文
- 重大资产清单

**6. 诉讼仲裁**
- 正在进行的诉讼/仲裁
- 已结案的重大案件
- 潜在法律风险

**准备步骤**：

**第一步：建立尽调资料室（Data Room）**
- 电子资料室：使用VDR（Virtual Data Room）系统
- 分类建立文件夹
- 设置访问权限（不同投资方不同权限）

**第二步：收集整理文件**
- 按照尽调清单逐一收集
- 缺失的文件及时补充
- 敏感信息脱敏处理

**第三步：问题预判与准备**
- 识别潜在法律问题
- 准备解释说明
- 准备解决方案

**常见法律问题及应对**：

| 问题 | 影响 | 应对 |
|-----|-----|-----|
| 股权代持 | 权属不清 | 解除代持，还原真实股权结构 |
| 知识产权瑕疵 | 核心资产风险 | 补充转让协议，完善权属 |
| 资质缺失 | 业务合规风险 | 尽快申请，披露时间表 |
| 重大合同纠纷 | 财务风险 | 准备诉讼方案，计提准备金 |
| 社保公积金欠缴 | 劳动风险 | 补缴计划，取得员工谅解 |

**尽调期间的注意事项**：

**1. 保密管理**
- 与投资方签署保密协议（NDA）
- 核心商业秘密控制披露范围
- 尽调结束后要求销毁或返还资料

**2. 陈述与保证**
- 投资方会要求创始人在协议中陈述保证
- 确保陈述内容真实准确
- 对不确定事项明确披露

**3. 交易文件谈判**

**关键条款**：
- 估值与投资额
- 股权结构（投前/投后）
- 优先权利：清算优先、反稀释、优先认购
- 保护条款：一票否决权、对赌条款
- 创始人限制：竞业禁止、股权锁定

**谈判策略**：
- 核心条款坚持（控制权、估值）
- 非核心条款可让步
- 寻求平衡，避免过度保护条款

**交割条件**：
- 法律尽调无重大不利发现
- 业务尽调满意
- 签署正式投资协议
- 完成工商变更

**法务在融资中的角色**：
1. 准备尽调资料
2. 参与交易文件谈判
3. 识别和提示法律风险
4. 协调外部律师
5. 办理交割手续

**时间规划**：
- 尽调准备：1-2周
- 尽调执行：2-4周
- 文件谈判：2-4周
- 交割：1-2周
- 总计：2-3个月
