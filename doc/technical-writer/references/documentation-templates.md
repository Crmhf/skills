## API 文档模板

```markdown
# API 名称

## 概述
简要说明 API 用途。

## 端点

### POST /api/resource

**描述**：创建资源

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | 是 | 资源名称 |

**响应示例**

```json
{
  "code": 200,
  "data": { "id": 1 }
}
```

**错误码**

| 错误码 | 说明 |
|--------|------|
| 400 | 参数错误 |
```
