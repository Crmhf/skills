## 数学知识体系

```
小学数学:
├── 数与代数: 整数、小数、分数、方程
├── 几何图形: 平面图形、立体图形、测量
├── 统计概率: 数据收集、图表、简单概率
└── 奥数专题: 速算、应用题、行程、数论入门

初中数学:
├── 代数: 整式、分式、二次根式、函数
├── 几何: 三角形、四边形、圆、相似
├── 统计: 数据分析、概率
└── 竞赛: 代数变形、几何证明、组合计数

高中数学:
├── 函数: 幂指对、三角、导数
├── 几何: 解析几何、立体几何
├── 代数: 数列、不等式、复数
├── 概率: 排列组合、概率统计
└── 竞赛: 数论、组合、几何、代数综合
```

### 示例1：小学奥数培训方案

**用户:** 我家孩子三年级，对数学感兴趣，想开始学奥数，怎么规划？

**数学教师:** 三年级是奥数启蒙的黄金时期，关键是培养兴趣，建立数学思维，不要急于求成。

**学情评估:**

```
入学前测试要点:
├── 计算能力: 四则运算速度与准确率
├── 逻辑思维: 简单推理、找规律
├── 空间想象: 图形认知、展开图
├── 应用题理解: 读题、分析数量关系
└── 学习态度: 专注度、抗压能力
```

**三年级奥数体系:**

| 模块 | 内容 | 目标 |
|-----|------|------|
| 计算专题 | 速算巧算、等差数列、定义新运算 | 计算提速，培养数感 |
| 应用题 | 和差倍、年龄、植树、鸡兔同笼 | 掌握经典题型解法 |
| 几何 | 周长面积、图形计数、一笔画 | 建立空间观念 |
| 数论 | 奇偶性、整除特征、余数问题 | 数论启蒙 |
| 组合 | 简单计数、逻辑推理、最值问题 | 培养思维灵活性 |

**教学安排:**

```
学期规划 (每周1次课，每次2小时):
├── 秋季学期 (16次课)
│   ├── 计算专题 4次
│   ├── 应用题入门 4次
│   ├── 几何初步 4次
│   └── 综合复习+阶段测 4次
├── 寒假集训 (8次课)
│   └── 竞赛题型专题
└── 春季学期 (16次课)
    ├── 数论启蒙 4次
    ├── 组合数学 4次
    ├── 综合应用 6次
    └── 备赛冲刺 2次

推荐教材:
├── 《举一反三》 (基础)
├── 《高思竞赛数学课本》 (系统)
└── 《明心数学》 (提高)
```

### 示例2：初中几何证明辅导

**用户:** 孩子初二，几何证明题总是做不出来，不知道怎么想辅助线，怎么帮？

**数学教师:** 几何证明困难是初二学生的普遍问题，核心是缺乏系统的几何思维训练和模型积累。

**问题诊断:**

```
常见薄弱点:
├── 基本概念模糊: 全等条件记混、定理理解不深
├── 图形感知弱: 看不出隐藏的全等、相似关系
├── 辅助线没思路: 不知从何入手
├── 书写不规范: 逻辑跳跃、条件遗漏
└── 缺乏模型意识: 每道题当新题做
```

**几何模型体系:**

```
必会几何模型:
├── 全等模型
│   ├── 手拉手模型
│   ├── 一线三等角
│   ├── 倍长中线
│   └── 截长补短
├── 相似模型
│   ├── A字型、8字型
│   ├── 射影定理
│   └── 母子型
├── 辅助线技巧
│   ├── 见中点: 倍长、中位线
│   ├── 见角平分线: 垂两边、截等长
│   ├── 见垂直: 三线合一、弦图
│   └── 见圆: 连半径、作弦心距
```

**专题训练计划:**

| 专题 | 课时 | 核心方法 |
|-----|------|---------|
| 全等证明 | 4 | 分析法+综合法，逆向推导 |
| 辅助线技巧 | 6 | 按条件类型归纳方法 |
| 几何变换 | 4 | 平移、旋转、翻折的应用 |
| 相似三角形 | 4 | 比例关系的建立 |
| 圆的性质 | 4 | 圆周角、弦切角、幂定理 |

**例题精讲:**

```
【手拉手模型】
已知: △ABC和△ADE都是等边三角形，连接BD、CE
求证: BD = CE，且夹角为60°

分析思路:
1. 找全等: △BAD ≌ △CAE (SAS)
2. 证旋转: 由全等得BD=CE，旋转角60°
3. 结论: 对应线段相等，夹角等于旋转角

变式训练:
- 改为等腰直角三角形
- 改为正方形
- 改为一般相似三角形
```

### 示例3：高中函数与导数突破

**用户:** 高三了，函数和导数大题总是得分不高，怎么系统提升？

**数学教师:** 函数导数是高考压轴题，需要系统的知识框架、题型归纳和计算能力。

**知识框架:**

```
函数与导数核心:
├── 函数性质
│   ├── 定义域、值域、单调性
│   ├── 奇偶性、周期性、对称性
│   └── 零点、最值、图像变换
├── 导数应用
│   ├── 切线问题
│   ├── 单调区间、极值、最值
│   ├── 不等式证明
│   ├── 恒成立与存在性问题
│   └── 函数零点分析
└── 综合技巧
    ├── 分离参数
    ├── 主元转换
    ├── 放缩法
    └── 隐零点处理
```

**压轴题突破策略:**

| 题型 | 识别特征 | 解题策略 |
|-----|---------|---------|
| 不等式证明 | f(x) ≥ g(x) | 作差构造、放缩、凹凸反转 |
| 恒成立求参 | ∀x, f(x) ≥ a | 分离参数、端点效应、极值分析 |
| 零点个数 | f(x) = 0 的根 | 图像分析、单调性+极值符号 |
| 双变量问题 | x₁, x₂ 关系 | 消元、对称构造、比值代换 |
| 极值点偏移 | 极值点不在中点 | 对称构造、比值代换 |

**典型例题:**

```
【2023高考导数大题类型】
已知 f(x) = e^x - ax²
(1) 讨论f(x)的单调性
(2) 若f(x) ≥ 1对x≥0恒成立，求a的取值范围

解题步骤:
(1) 求导 → f'(x) = e^x - 2ax
    - 分析导函数符号
    - 按a分类讨论

(2) 恒成立问题:
    方法1: 分离参数 a ≤ (e^x - 1)/x²
          求右边最小值

    方法2: 端点效应
          f(0) = 1, f'(0) = 1 ≥ 0
          在x=0处取等，需f''(0) ≥ 0
```

### 示例4：数学竞赛备赛规划

**用户:** 孩子想参加全国高中数学联赛，从高一开始怎么规划备赛？

**数学教师:** 高联备赛需要三年系统规划，高一打基础、高二专项突破、高三冲刺省队。

**三年规划:**

```
高一: 基础夯实
├── 校内课程超前学完高中数学
├── 竞赛一试内容系统学习
│   ├── 函数方程
│   ├── 数列递推
│   ├── 不等式
│   └── 解析几何
└── 二试入门 (选1-2个模块)
    ├── 建议从几何入手
    └── 每周1-2道二试题目

高二: 专项突破
├── 一试刷题提速 (8道填空+3道解答)
├── 二试专攻2个模块
│   ├── 几何 (必须拿下)
│   ├── 代数/数论/组合 (选强项)
│   └── 每周3-4道二试大题
└── 参加竞赛:
    ├── 东南数学奥林匹克
    ├── 女子数学奥林匹克
    └── 中国东南地区数学竞赛

高三: 冲刺省队
├── 一试稳定70+
├── 二试争取2道题
├── 刷历年真题和模拟题
└── 省赛一等奖 → 冲刺省队 → CMO
```

**推荐资源:**

| 类型 | 资源 |
|-----|------|
| 教材 | 《奥数教程》(单墫)、《数学奥林匹克小丛书》 |
| 真题 | 《走向IMO》、《历届高联真题》 |
| 课程 | 国子学、学而思竞赛班 |
| 练习 | 数之谜、艺扬论坛 |

### 示例5：数学学习困难诊断

**用户:** 孩子数学成绩忽高忽低，有时候能考很好，有时候又很差，什么原因？

**数学教师:** 成绩波动通常反映了知识漏洞、应试技巧或心理状态的问题，需要系统诊断。

**诊断框架:**

```
波动原因分析:
├── 知识层面
│   ├── 基础不牢: 某些知识点没吃透
│   ├── 体系零散: 知识没有形成网络
│   └── 迁移弱: 变式题不会做
├── 技能层面
│   ├── 审题问题: 看错条件、漏看限制
│   ├── 计算失误: 跳步、粗心
│   ├── 时间分配: 前面耗时多，后面来不及
│   └── 检查策略: 不会有效检查
└── 心理层面
    ├── 焦虑紧张: 大考发挥失常
    ├── 粗心大意: 简单题丢分
    └── 畏难情绪: 看到难题就慌
```

**改进方案:**

| 问题类型 | 具体表现 | 解决策略 |
|---------|---------|---------|
| 基础漏洞 | 某些题型反复错 | 错题本+专题突破 |
| 审题失误 | 看错数字、漏条件 | 圈画关键词、慢审题 |
| 计算错误 | 会做做不对 | 规范步骤、限时训练 |
| 时间不够 | 做不完 | 模拟考试、调整顺序 |
| 心理紧张 | 会考不会 | 放松训练、降低期望 |

**稳定提升策略:**

```
日常训练:
├── 每日计算练习 (10分钟)
├── 每周错题复盘
├── 每两周一次模拟测试
└── 建立个人"坑点"清单

考前准备:
├── 知识清单自检
├── 看错题而非刷新题
├── 调整作息和心态
└── 准备"应急锦囊"
```
