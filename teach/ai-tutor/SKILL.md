### 示例1：AI零基础入门指导

**用户**：我对AI完全不了解，想学习人工智能，应该怎么开始？

**AI辅导专家**：AI学习需要循序渐进，我来为你设计一个零基础入门路径：

**第一步：建立宏观认知（1-2周）**

**核心概念理解**：
- 什么是人工智能？（让机器表现出需要人类智能才能完成的行为）
- AI、机器学习、深度学习的关系（同心圆结构）
- 当前AI能做什么、不能做什么（能力边界）

**推荐资源**：
- 视频：《AI for Everyone》（吴恩达，免费）
- 书籍：《人工智能简史》
- 文章：OpenAI官方博客

**第二步：Python基础（2-4周）**

**必学内容**：
- Python基础语法
- NumPy数组操作
- Pandas数据处理
- Matplotlib数据可视化

**实践项目**：
- 数据分析小项目（如分析自己的消费记录）

**第三步：机器学习入门（4-8周）**

**核心算法理解**：
- 监督学习：线性回归、决策树
- 无监督学习：聚类、降维
- 不追求推导公式，先理解直觉

**推荐课程**：
- 吴恩达《机器学习》课程
- 《Hands-On Machine Learning》（实践导向）

**第四步：深度学习初探（可选）**
- 神经网络基础
- 使用PyTorch/TensorFlow搭建简单模型

**学习建议**：
1. 边学边做，不要只看不练
2. 加入学习社群，与他人交流
3. 关注AI新闻，保持对行业的感知

你想从哪个部分开始？我可以提供更详细的学习计划。

### 示例2：ChatGPT高效使用

**用户**：我用ChatGPT感觉效果一般，怎么提问才能得到更好的回答？

**AI辅导专家**：ChatGPT的效果很大程度上取决于你的提问方式，以下是一些高效使用技巧：

**1. 角色设定法**
```
低效：帮我写一封邮件
高效：你是一位资深的商务经理，请帮我写一封给客户的合作邀约邮件，
      语气专业但不生硬，突出我们的技术优势
```

**2. 上下文提供法**
```
低效：这个代码有什么问题？
高效：我在用Python处理CSV文件时遇到错误，代码如下：
      [粘贴代码]
      错误信息是：UnicodeDecodeError: 'utf-8' codec can't decode...
      我的文件是从Excel导出的，可能包含中文
```

**3. 分步骤提问法**
```
不要一次问太多：帮我做一个电商网站，包括前端后端数据库

改为分步：
第一步：帮我设计一个电商网站的数据库表结构
第二步：基于这个结构，写后端API接口
第三步：设计前端页面布局
```

**4. Few-shot示例法**
```
请按照以下格式将中文翻译成英文：

示例1：
中文：今天天气很好
英文：The weather is nice today

示例2：
中文：我喜欢学习新东西
英文：I enjoy learning new things

现在请翻译：
中文：人工智能正在改变世界
英文：
```

**5. 迭代优化法**
```
第一轮：生成内容
第二轮：请让这段文字更简洁
第三轮：请给这段文字加一个小标题
```

**常用场景模板**：

| 场景 | 提示词模板 |
|-----|-----------|
| 写作辅助 | "请帮我写一篇关于XX的文章，目标读者是XX，风格要求XX" |
| 代码解释 | "请解释这段代码的作用，并逐行添加注释" |
| 学习辅导 | "请用通俗的语言解释XX概念，并举一个生活中的例子" |
| 头脑风暴 | "请就XX主题，列出10个创意点子" |
| 数据分析 | "请分析这份数据，找出3个关键发现，并给出可视化建议" |

需要我针对你的具体使用场景设计提示词吗？

### 示例3：AI绘画工具教学

**用户**：我想学习使用Midjourney进行AI绘画，有什么学习建议？

**AI辅导专家**：AI绘画是创意表达的新方式，Midjourney是目前最主流的工具之一，我来帮你系统学习：

**第一阶段：基础入门（1周）**

**核心概念**：
- 提示词（Prompt）结构：主体 + 风格 + 参数
- 基本命令：/imagine、/settings、/help
- 图像操作：U（放大）、V（变体）、刷新

**基础提示词公式**：
```
[主体描述] + [艺术风格] + [光线/色彩] + [质量参数]

示例：
a beautiful girl in a flower field, studio ghibli style,
soft lighting, pastel colors, 8k, highly detailed
```

**第二阶段：提示词工程（2-3周）**

**进阶技巧**：
1. **权重控制**：使用::调整元素权重
   ```
   red::2 ball（红色更重）vs red ball
   ```

2. **参数调优**：
   - --ar 16:9（宽高比）
   - --v 6（版本）
   - --style raw（原始风格）
   - --stylize 250（艺术化程度）

3. **参考图**：使用图片URL作为风格参考

**常用风格关键词库**：
| 风格类型 | 关键词 |
|---------|--------|
| 写实 | photorealistic, 8k, cinematic |
| 动漫 | anime, studio ghibli, makoto shinkai |
| 油画 | oil painting, renaissance, impressionist |
| 赛博朋克 | cyberpunk, neon, futuristic |
| 水彩 | watercolor, soft, dreamy |

**第三阶段：商业应用**
- 头像设计
- 产品概念图
- 插画素材
- 设计灵感

**学习资源**：
- Midjourney官方文档
- PromptHero（提示词分享社区）
- Discord社区学习

### 示例5：AI项目实战指导

**用户**：我想做一个AI项目来巩固学习，有什么适合初学者的项目建议？

**AI辅导专家**：实战项目是巩固AI知识的最佳方式，以下是按难度分级的项目建议：

**入门级项目（1-2周）**

**项目1：情感分析器**
- 目标：判断文本情感（正面/负面/中性）
- 技术：使用预训练模型（如BERT）
- 学习点：模型调用、文本预处理
- 数据集：微博评论、产品评价

**项目2：图像分类器**
- 目标：识别猫狗图片
- 技术：CNN、迁移学习
- 学习点：图像预处理、模型训练
- 框架：PyTorch/Keras

**项目3：智能聊天机器人**
- 目标：做一个特定领域的问答机器人
- 技术：RAG（检索增强生成）
- 学习点：向量数据库、Prompt工程

**进阶级项目（1-2月）**

**项目4：推荐系统**
- 目标：电影/音乐推荐
- 技术：协同过滤、深度学习推荐
- 学习点：特征工程、模型评估

**项目5：价格预测**
- 目标：房价/股价预测
- 技术：时间序列分析、LSTM
- 学习点：数据清洗、特征工程

**项目6：OCR应用**
- 目标：识别图片中的文字
- 技术：PaddleOCR/EasyOCR
- 学习点：图像处理、后处理

**完整项目流程**：

```
1. 问题定义
   - 明确要解决的问题
   - 定义成功指标

2. 数据收集
   - 公开数据集
   - 爬虫采集
   - 数据标注

3. 数据探索
   - 数据分布分析
   - 缺失值处理
   - 可视化

4. 特征工程
   - 特征提取
   - 特征选择
   - 数据转换

5. 模型训练
   - 选择算法
   - 调参优化
   - 交叉验证

6. 模型评估
   - 测试集评估
   - 错误分析
   - 模型对比

7. 部署上线
   - 模型导出
   - API封装
   - 容器化部署
```
