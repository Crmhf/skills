## 定性研究

### 用户访谈

| 类型 | 时长 | 样本量 | 适用场景 |
|------|------|--------|----------|
| 结构化 | 30-60min | 5-10人 | 验证假设 |
| 半结构化 | 45-90min | 8-15人 | 探索性研究 |
| 情境访谈 | 60-120min | 3-5人 | 深度理解 |

### 可用性测试

```
测试流程:
1. 介绍 (5min)
2. 背景问卷 (5min)
3. 任务执行 (30-45min)
4. 后测访谈 (10min)
5. SUS量表 (5min)
```

## 定量研究

### 问卷调查

| 题型 | 用途 | 示例 |
|------|------|------|
| 单选题 | 分类统计 | 您最常用的功能是？ |
| 量表题 | 满意度测量 | NPS评分 1-10 |
| 多选题 | 多因素分析 | 您常用的功能有？ |
| 开放题 | 收集反馈 | 您有什么建议？ |

### A/B 测试

```
测试设计:
- 假设: 红色按钮比蓝色按钮点击率高
- 样本量: 每组至少1000用户
- 周期: 2周
- 指标: 点击率、转化率
```

## 研究流程

```
1. 明确目标 → 要回答什么问题？
2. 选择方法 → 定性/定量/混合
3. 招募用户 → 样本量、筛选条件
4. 执行研究 → 数据收集
5. 分析洞察 → 模式识别
6. 输出报告 → 行动建议
```
